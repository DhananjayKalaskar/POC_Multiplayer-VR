
# Unity Multiplayer VR GameUnity Multiplayer VR Game using Unity Netcode for networking, Vivox for voice chat, and XR Interaction Toolkit with OpenXR.## Features- VR player rig with synced head and hand tracking- Multiplayer support via Unity Netcode for GameObjects- Automatic voice chat between connected players using Vivox- XR Interaction Toolkit input and locomotion- ParrelSync-ready workflow for local multiplayer testing (optional)

## Requirements- Unity 2022.3.62f2 (matching ProjectVersion.txt)- Android / PC VR build support installed (depending on your target)- Vivox credentials configured under `Edit > Project Settings > Services > Vivox`- Microphone permission enabled on the target device
